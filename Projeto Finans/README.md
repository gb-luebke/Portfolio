# Finans

**Descrição:**
 - Este projeto é uma simulação do design de um site de finanças. O site é totalmente responsivo. As funcionalidades do site não estão programadas, apenas o design. 
 
 **Tecnologias:**
 - HTML
 - CSS
 - Bootstrap
