# Calculadora

**Descrição:**
 - Este projeto consiste em uma calculadora básica online. A calculadora possui as mesmas funcionalidades da calculadora básica do Iphone.
 
 **Tecnologias:**
 - O projeto foi criado utilizando a estrutura base do react.
 - Biblioteca mathjs
 - Hooks: useState
 - React Icons
